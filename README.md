# bedrock-java-sdk-demo

## pre-requisites
1. configure pom.xml @dev host
2. configure aws account credential @dev host
```sh
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install
```
